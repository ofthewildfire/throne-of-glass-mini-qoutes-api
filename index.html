Simple Throne of Glass Qoute API

Qoutes from the fantasy novel series Throne of Glass by Sarah. J Maas

Its crazy tiny, like, only my faves tiny!

Endpoints

For all qoutes:

https://throneofglass.cyclic.app/quotes/

Qoutes are numbered starting at one (1), this will get you the first qoute in the list

https://throneofglass.cyclic.app/1

If there is no qoute for the ID you search you will return an error object telling you there isn't a quote! Likely means you've run out of my silly qoutes! :)

{error: 'No qoute'}
